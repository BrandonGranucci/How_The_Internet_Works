Domain Name System

  • Domain Name Systems have the responsibility of making websites publicly available
  • DNS's are broken into numerous hierarchies and machines that allow user to access websites
  • The most popular top level domain consists of .com, .ru, .org, and .net
  • The Internet relies off these servers to allow for users to visit websites from anywhere across the world
